🔑 Google Cloud TTS API 키 설정

Google Cloud TTS를 사용하면 화자별 실제 다른 목소리가 적용됩니다.
Tim, Dad, Mary, Kit, Harry 등 각 캐릭터마다 다른 영국식 Neural2 음성이 사용됩니다.

📋 API 키 발급 방법
  1. console.cloud.google.com 접속 → 프로젝트 생성
  2. 검색창에 "Text-to-Speech API" 검색 → 사용 설정
  3. API 및 서비스 → 사용자 인증 정보 → API 키 만들기
  4. 생성된 키 복사 (형식: AIzaSy...)

💡 월 100만 자 무료 (Neural2 음성 포함)

🔑 Google TTS 키: ⚪ 미설정
🇬🇧 영국식 vs 🇺🇸 미국식 비교
water / butter / better / letter
🇬🇧 /ˈwɔːtə/ /ˈbʌtə/ /ˈbetə/ /ˈletə/  |  🇺🇸 /ˈwɑːt̬ɚ/ /ˈbʌt̬ɚ/ /ˈbet̬ɚ/ /ˈlet̬ɚ/
💡 미국식: 't'가 모음 사이에서 /r/ 비슷한 탄음으로 변함 (버러, 워러). 영국식: 't'를 명확히 발음하고 'r'은 묵음 (버터, 워터).
can't / dance / bath / path / ask
🇬🇧 /kɑːnt/ /dɑːns/ /bɑːθ/ /pɑːθ/ /ɑːsk/  |  🇺🇸 /kænt/ /dæns/ /bæθ/ /pæθ/ /æsk/
💡 영국식 RP: 'a'가 /ɑː/ (입을 크게 열어 '아아'). 미국식: /æ/ (앞니를 보이며 '애').
